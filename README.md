# MJ Tech Hub Website

Production website for MJ Tech Hub — an integrated IT, network, security, VoIP and AI solutions provider serving Toronto and the GTA.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Nodemailer quote workflow

## Primary routes

- `/` — company overview and core solutions
- `/services` — integrated service capabilities
- `/voip` — business phone systems
- `/ai-receptionist` — AI phone and appointment workflows
- `/office-setup` — complete business technology deployments
- `/residential` — home network, security and smart technology
- `/projects` — representative project portfolio
- `/contact` — quote and support request

## Local development

```bash
npm install
npm run dev
npm run build
```
